Chessboard

My second artifact was to create a chessboard using mainly CSS. This artifact consists of three main parts, which are header, the chessboard, and footer. One essential rule for creating this program was to NOT use a table element to create the board. All the rows of the board were created using external CSS, and no internal style element was used. Also, I utilized the <div> tag, which is a division element, along with the class attribute for styling the board.
