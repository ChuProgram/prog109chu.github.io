Chessboard

My second artifact was to create a chessboard using mainly CSS. This artifact consists of three main parts, which are header, the chessboard, and footer. One essential rule for creating this program was to NOT use a table element to create the board. All the rows of the board were created using external CSS, and no internal style element was used. Also, I utilized the <div> tag, which is a division element, along with the class attribute for styling the board.

Rhombus

   The third artifact I designed was creating a rhombus by utilizing all three languages: HTML, CSS, and JavaScript. This HTML page ask a user, using HTML input elements (instead of using prompt method), that asks following to create a rhombus (diamond) shape...

  • Height of each section of the rhombus
  • Color for the ODD position
  • Color for the EVEN position
  • Symbol to be used to draw the rhombus

Contact Us Form

The fourth artifact done in the class was creating a "Contact Us" form using JavaScript. The purpose of using JavaScript was to validate the user inputs on the form. For instance, if user did not put anything on First Name box, then the website would show an error message telling the user that this field is required. Thus, the JavaScript would validate user inputs on the page.
